Class Color

Color in RGBA format with some support for ARGB format. A color is represented by red, green, and blue (r, g, b) components. Additionally, a represents the alpha component, often used for transparency. Values are in floating-point and usually range from 0 to 1. Some properties (such as CanvasItem.modulate) may accept values greater than 1.

You can also create a color from standardized color names by using @GDScript.ColorN.

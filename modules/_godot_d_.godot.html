abandon_value

  • abandon_value(value: any): number
    • Defined in godot.d.ts:211

    Drop the value in the context. You should never touching an abandoned value anymore before it is adopted.

    You can adopt the value in another thread context to transfer the value in threads.

    A non zero ID is return if no error happen.

    note

    You can only transfer values the godot Variant can represent.

    Parameters

    • value: any

      The value to abandon in the context

    Returns number

    The ID of the abandoned value

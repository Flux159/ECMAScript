FrameRequetID

FrameRequetID: number
  • Defined in godot.d.ts:8

A long integer value, the request id, that uniquely identifies the entry in the callback list. This is a non-zero value, but you may not make any other assumptions about its value. You can pass this value to godot.cancelAnimationFrame() to cancel the refresh callback request.
